Iconography — rounded outline, stroke-width 2

Handoff notes: Font is Manrope (Google Fonts) as a web-safe substitute for SF Pro Display. All icons are hand-drawn inline SVG at stroke-width 2 — swap for the Lucide icon set 1:1 by name in production. Radii and spacing follow an 8pt grid throughout. Gradients run 120°–160° diagonal. Card background never uses pure black — always #161616 — to keep depth against the #0A0A0A base.
